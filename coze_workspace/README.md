# 治愈系漫画工作流(coze 复刻版)

## 这是什么?

参考微信公众号文章「扣子(coze):温暖治愈系漫画,工作流制作生成,还可用于打造个人IP」写的复刻版。

**原文是付费的** — 作者要求"一键三联+关注"才给完整系统提示词。这个仓库是我的开源复刻版。

## 文件清单

| 文件 | 用途 |
|------|------|
| `prompts.md` | 4 段系统提示词 + 调用参数(直接复制粘贴到 coze 节点) |
| `workflow_setup_guide.md` | coze 界面手动搭建步骤(每个节点 1-2 分钟) |
| `healing_comic_workflow.json` | 工作流 JSON DSL 尝试版(可尝试导入) |
| `README.md` | 本文件 |

## 限制

- image2 是付费模型,跑一次约 0.5-2 元(4 张图)
- 我的提示词是复刻,质量可能略差于原文
- JSON DSL 是基于通用格式猜的,coze 实际 DSL 可能有差异,不一定能直接导入

## 怎么用?

### 方式 A: 手动搭建(推荐,稳)

1. 按 `workflow_setup_guide.md` 步骤在 coze 界面拖拽节点
2. 每个节点的提示词从 `prompts.md` 复制粘贴
3. 试运行输入"下班后"

### 方式 B: 导入 JSON(快,但可能需要微调)

1. 进入 coze 资源库 → 创建工作流
2. 找"导入"或"导入 JSON"按钮(可能藏在右上角"...")
3. 上传 `healing_comic_workflow.json`
4. 修正任何红色错误提示(节点 ID/变量名可能不匹配)
5. 试运行

## 工作流结构

```
开始(title, key)
   ├─ title → LLM-治愈文案 → texts(JSON) → str_to_list → text_list ─┐
   │                                                              ↓
   └─ key   → image2-参考图 → ref_image ────────────────→ 批处理   │
                                                              ↓
                                              ┌─── LLM-图片 prompt
                                              └─── image2-漫画 (4张)
                                                              ↓
                                                       image_urls
                                                              ↓
                                                          结束.images
```
