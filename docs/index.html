🟡 银行行长 30 年 · 游戏说明 📄 Markdown 源文件 🎮 玩游戏

玩家角色:1995 年 22 岁入行小柜员 · 基调:派系 + 监管 + 行业抉择 · 时长:30—45 分钟

🎬 开局状态(1995)

你 22 岁,刚毕业,父母 50 岁。
分配到中国工商银行某市分行支行,做柜员。

【硬指标】
  存款规模       500 万
  不良率         0%
  监管评级       3A(顶级)
  政治资本       20

【软指标】
  Health(健康)   100
  Family(家庭)   100
  Social(社交)   80

📊 数值体系

硬指标起点影响
存款规模500 万升职速度(行内排名)
不良率0%监管评级(越低越好)
监管评级3A能否升分行行长
政治资本20站队/派系生存

软指标致命线 = 30:Health 过劳病退 / Family 家人决裂 / Social 边缘化

🕐 五个时代

  • 1995—2000 利率管制:四大行垄断 + 坏账隐现(WTO 谈判、央行高息揽存)
  • 2000—2005 股份制崛起:招商/民生/兴业 入世 + 坏账剥离 + 信用卡大战
  • 2005—2013 黄金十年:房地产 + 城投 + 表外(4 万亿、影子银行)
  • 2013—2018 互联网金融:余额宝/P2P/移动支付 + 资管新规
  • 2018—2025 监管周期:200+ 行长被查 + 数字人民币 + AI 风控

🎯 每回合 3 步决策

  1. 时代事件(2 选 1):影响软指标 + 政治资本(例:2001 跳股份制银行?)
  2. 业务项目(3 选 1):影响硬指标(例:2003 信用卡 / 代销基金 / 银保通)
  3. 投入度(3 选 1):A 996 极限 / B 8h 平衡 / C 摆烂
  4. 派系站队(2 选 1,每 4 回合):A 总行派(政治 +5, Family -3)/ B 地方派(Social +5, 政治 -3)

🚪 时代门控(升职硬指标)

过渡硬要求否则
1 → 2存款 ≥ 5 亿卡在客户经理
2 → 3监管 ≥ 2A下放郊区支行
3 → 4政治 ≥ 502018 反腐第一批被查
4 → 5存款 ≥ 100 亿卡在支行行长退休

🏆 9 结局 + 2 隐藏

  • 🏛 国有大行总行行长:监管 3A + 存款 1000 亿+ + 政治 80+
  • 💼 股份制分行行长:监管 2A + 存款 500 亿+
  • 🏙 城商行总行行长:监管 2A + 存款 100 亿+
  • 🏡 稳定中产支行长:监管 1A + 存款 50 亿+
  • ⚠️ 35 岁被裁:监管 3C + 政治 < 40
  • 🚨 金融反腐被查:政治 ≤ 30(任意时代)
  • 💔 家人决裂:Family ≤ 30
  • 🔒 隐藏:民营银行 VP:时代 4 跳民营 + 存款 100 亿+
  • 🔒 隐藏:逃过一劫:政治 ≥ 90 + 监管 3A

💡 关键选择提示

"功成身退"路线核心策略:

  1. 存款是升职门票,投入度选 A 拉存款,但 Health 消耗快,需要 Family ≥ 60
  2. 政治资本是反查护身符,派系选"总行派"涨最快
  3. 时代 1 → 2 需要存款 5 亿,选揽存大战 / 政策性贷款
  4. 时代 3 → 4 需要政治 ≥ 50,否则 2018 第一批被查
  5. 不要站错队,时代 4 跳股份制政治会掉但存款涨,平衡

首玩推荐路径:稳定中产支行长(最容易达成):

  • 投入度基本选 B(8h 平衡)
  • 派系看哪个 Family 高选哪个
  • 业务:时代 1 揽存大战 / 时代 2 信用卡 / 时代 3 稳健项目 / 时代 4 合规优先
  • 政治保持 30—50 之间

🟢 基金经理 30 年 · 游戏说明 📄 Markdown 源文件 🎮 玩游戏

玩家角色:1995 年 22 岁入行基金研究员 · 基调:穿越中国资本市场五时代 · 时长:30—45 分钟

🎬 开局状态(1995)

你 22 岁,刚从复旦/清华金融系毕业。
加入君安证券(还没和申银合并),做行业研究员。

【硬指标】
  AUM(管理规模)     0
  收益率             0%
  排名百分位         100%(垫底)
  老鼠仓风险         0

【软指标】
  Health      100
  Family      100
  Social      80
  Reputation  60

📊 硬指标

指标影响
AUM(管理规模)升职 / 跳槽筹码
收益率业绩基准
排名百分位业绩排名(越小越好)
老鼠仓风险监管 / 牢狱风险

🕐 五个时代

  • 1995—2000 庄股时代:君安/申银/万科/吕梁/德隆
  • 2000—2005 价值投资启蒙:第一波价值派 + QFII + 熊市
  • 2005—2013 公募黄金期:6124 大牛市 + 王亚伟 + 余额宝
  • 2013—2018 私募崛起:公奔私 + 阳光私募 + 资管新规
  • 2018—2025 量化 AI:幻方/九坤/明汯/DeepSeek

🏆 9 结局 + 2 隐藏

  • 🏛 公募基金总经理:AUM 500 亿+ + 排名 < 30%
  • 💼 私募大佬(百亿+):AUM 100 亿+ + 收益 ≥ 20%
  • 🌐 量化私募合伙人:时代 5 + 量化 + 收益 30%+
  • 📚 价值投资大师:时代 2 起持续价值派 + 长期收益
  • ⚠️ 35 岁被淘汰:排名长期 > 80%
  • 🚨 老鼠仓案发:老鼠仓 ≥ 80(可能坐牢)
  • 💔 家庭破裂:Family ≤ 30
  • 🔒 隐藏:公奔私成功:时代 4 跳私募 + AUM 100 亿+
  • 🔒 隐藏:AI 量化之神:时代 5 押注 AI + 收益 50%+

💡 关键选择提示

"公募基金总经理"路线(稳扎稳打):

  1. 时代 1(庄股)选价值投资,为后续打好声誉基础
  2. 时代 2/3 不要追高,6124 大牛市前保持仓位灵活
  3. 时代 3(公募黄金期)全力扩 AUM,这是基金行业最容易做大的时代
  4. 老鼠仓风险控制在 30 以下,否则随时可能被查
  5. Family 维持 60+,长期过劳容易导致家庭破裂

⚠️ 老鼠仓陷阱:老鼠仓 ≥ 60 进入被监管注意频发期,≥ 80 直接案发

  Family      100
  Social      80

🕐 四个时代

  • 2005—2010 BAT 创业潮:阿里 / 腾讯 / 百度
  • 2010—2015 移动互联网:iPhone / 安卓 / App 创业 / O2O
  • 2015—2020 资本寒冬:区块链泡沫 / 996 / P2P 崩盘
  • 2020—2025 AI Agent:GPT / Copilot / DeepSeek

🏆 7 结局 + 2 隐藏

  • 🏛 顶级架构师:Coding 90+ + Rep 70+
  • 💼 技术合伙人:Coding 70+ + Insight 80+ + 时代 4 AI
  • 🚀 创业成功(独角兽):Insight 90+ + Net 80+
  • 🌐 大厂 P9/P10:Rep 80+ + Coding 80+

  • 🚨 过劳猝死:Health ≤ 30(最惨)
  • 💔 家人决裂:Family ≤ 30
  • 🔒 隐藏:开源大神:Rep 90+ + 开源项目主导
  • 🔒 隐藏:AI 教父:时代 4 押注 AI Native + Insight 95+

💡 关键选择提示

"大厂 P9/P10"路线(稳扎稳打):

  1. 时代 1 选加入大厂,Coding 涨得快,Rep 也涨
  2. 时代 2 选移动端,iOS/安卓 是 2010—2015 最吃香
  3. 时代 3 稳为主,别跳来跳去
  4. 时代 4 必须上车 AI,Coding + AI 是最优组合
  5. Health 维持 60+,长期 996 容易跌到致命线

⚠️ 致命警示:Health ≤ 30 → 过劳猝死(最惨结局,务必避免)
